This C++ program implements a simple Student Management System using a singly linked list. It allows users to add student records, display student information, and delete students based on their roll number through a menu-driven interface. Student data such as name, roll number, and marks are stored dynamically using pointers and linked list nodes. The project helps in understanding linked list operations, dynamic memory allocation, structures, pointers, and basic data management techniques in C++.
Technologies Used: C++, Linked Lists, Pointers, Structures, Dynamic Memory Allocation, Menu-Driven Programming
Features:
* Add student records
* Display student details
* Delete student records by roll number
* Dynamic memory management using linked lists
